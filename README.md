# 🚗 猜车标

基于 387 个汽车品牌的车标识别 H5 小游戏，支持限时挑战、无尽模式、十题冲刺、学习模式。

**在线玩：** https://你的用户名.github.io/car-guess-game

## 游戏模式

- ⏱️ **限时挑战**：60秒内答对越多越好
- ♾️ **无尽模式**：答错为止
- 🎯 **十题冲刺**：10题全对，用时最短获胜
- 📖 **学习模式**：无压力认识车标

## 部署

游戏数据直接嵌入 HTML，零依赖，双击 `index.html` 即可本地运行，或部署到 GitHub Pages。

```bash
# 推送到 GitHub Pages
git push -u origin main
```
